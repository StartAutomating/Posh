System.Management.Automation.PSModuleInfo.Dismount()
----------------------------------------------------

### Synopsis
Dismounts a module as a drive

---

### Description

Dismounts a module as a drive.

---

### Parameters
#### **DriveName**
The name of the drive.  If not provided, this will be the module.

|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[String]`|false   |1       |false        |
