Posh.Host.set_Title()
---------------------




### Synopsis
Sets the host's title



---


### Description

Sets the host's window title.

Unfortunately, this will be ignored if running in Visual Studio Code.



---


### Examples
#### EXAMPLE 1
```PowerShell
$Posh.Title = "Sup?"
```



---


### Parameters
#### **Title**




|Type      |Required|Position|PipelineInput|
|----------|--------|--------|-------------|
|`[String]`|false   |1       |false        |
