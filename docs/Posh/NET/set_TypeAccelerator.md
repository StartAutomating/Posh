Posh.NET.set_TypeAccelerator()
------------------------------

### Synopsis
Sets type accelerators

---

### Description

Sets type accelerators (aliases for types)

---

### Examples
> EXAMPLE 1

```PowerShell
$posh.Net.TypeAccelerator = @{'t'=[type]}
```
> EXAMPLE 2

```PowerShell
$posh.Net.Alias = 't10', [type]
```
